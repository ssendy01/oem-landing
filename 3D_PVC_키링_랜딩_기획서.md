## 1. 사업모델 3안 비교 + 추천 1안

| 구분 | A) 100% 견적형 | B) 패키지형 + 옵션견적 | C) 유료 단계형 (추천) |
|------|---------------|---------------------|---------------------|
| **핵심 메커니즘** | 문의 → 견적 → 본주문 | 표준 패키지 노출 → 옵션 추가 → 본주문 | 문의 → 샘플 진행비/우선권 결제 → 본주문 |
| **장점** | - 진입장벽 최소<br>- 리드 수 극대화<br>- 유연한 대응 가능 | - 가격 투명성<br>- 결정 속도 향상<br>- 운영 효율화 | - **고의향 고객 필터링**<br>- 샘플/상담 비용 회수<br>- 본주문 전환율 최고<br>- 무성의 문의 차단 |
| **단점** | - 냉문의 과다<br>- 견적 리소스 과다<br>- 전환율 낮음 | - 사양 변수 많아 패키지화 어려움<br>- 고객 니즈 불일치 가능<br>- 후기 없으면 결정 어려움 | - 초기 진입장벽<br>- 신뢰 구축 필수<br>- 결제 시스템 필요 |
| **운영 난이도** | 중 (견적 응대 부담) | 중~상 (패키지 설계/업데이트) | 하 (고의향만 집중 대응) |
| **분쟁 리스크** | 중 (무료 상담 → 책임범위 모호) | 중 (패키지 불만족 시) | **하** (결제 단계에서 계약 조건 명확화) |
| **전환 장벽** | 낮음 → 리드만 많고 전환 낮음 | 중 (가격 부담 + 사양 불일치) | 중~높음 → 하지만 **결제자는 고전환** |
| **추천 타깃** | 초기 인지도 확보 단계 | 표준화 가능한 상품 | **후기 적어도 신뢰 시스템으로 고의향 집객** |
| **권장 문구 톤** | "무료 견적 받기" | "500개 스타터 패키지 ₩XX~" | "샘플 진행비 ₩50,000 (본주문 시 전액 차감)" |

### 추천: **C) 유료 단계형**

#### 선택 이유
1. **후기 부족 → 고의향 고객에 집중하는 전략이 효율적**
   - 냉문의 대응에 시간 낭비하지 않고, 진지한 고객에게만 리소스 투입
2. **OEM 특성상 사양 변수 多 → 패키지화 불가능**
   - 사이즈/디자인/수량/포장이 모두 달라 표준 패키지는 비현실적
3. **신뢰 구축 방법론이 명확**
   - "샘플 진행비"는 단순 결제가 아니라 **"사양 확정 + 커뮤니케이션 착수 + 샘플 제작"의 가치를 전달하는 장치**
   - 결제 전 "프로세스 공개 + 검수 기준 + 사례 + FAQ"로 불안 제거
4. **본주문 전환율이 압도적**
   - 무료 견적형: 100명 문의 → 5명 주문 (5%)
   - 유료 단계형: 10명 결제 → 8명 주문 (80%)

#### 운영 체크리스트
- [ ] 리틀리 결제 링크 생성 (샘플 진행비 ₩50,000~₩100,000)
- [ ] 결제 전 명시사항 작성
  - "샘플 진행비는 사양 확정/3D 샘플 제작/우선 일정 홀드 비용입니다"
  - "본주문 진행 시 전액 차감됩니다"
  - "샘플 승인 후 본주문 안 할 경우, 환불은 불가하지만 샘플은 배송해드립니다"
- [ ] FAQ에 "샘플 진행비 왜 받나요?" 항목 추가
- [ ] CTA 배치: 메인(무료 폼) / 보조(메일) / 고의향(결제)
- [ ] 결제 CTA는 "진지하게 준비 중인 분"임을 명시하는 배지/문구로 포지셔닝
- [ ] 자동응답: 결제 완료 즉시 "48시간 내 1차 상담 연락드립니다" 메일 발송

1. **Q. 납기를 못 맞추면 어떻게 되나요?**
   A. 표준 리드타임은 50일 기준이며, 각 단계별로 진행률을 실시간 공유합니다. 통관은 세관 검사 등으로 ±5일 변동 가능하며, 지연 시 즉시 안내드립니다. 긴급 납기 원하시면 별도 상담 가능합니다.

2. **Q. 불량품이 나오면 재작업 가능한가요?**
   A. 네. 불량률 5% 이하를 보장하며, 초과분은 재작업합니다. 검수 기준 10개 항목을 단계별로 체크하고 사진으로 전송해 드립니다.

3. **Q. 통관은 제가 직접 해야 하나요?**
   A. 아니요. 통관 대행까지 포함되어 있습니다. 국내 배송지까지 배송해 드립니다.

4. **Q. 캐릭터/로고 지재권 문제 없나요?**
   A. 본인 창작물이거나 라이선스를 보유한 경우에만 제작 가능합니다. 2차 창작 / 유명 캐릭터는 라이선스 확인 후 진행합니다.

5. **Q. 선금 보내고 잠수 타는 거 아닌가요?**
   A. 샘플 진행비(우선 진행권)는 샘플 제작 착수 비용이며, 본주문 시 전액 차감됩니다. 각 단계마다 진행 상황을 공유하며, 연락 두절은 없습니다.

6. **Q. 환불 가능한가요?**
   A. 샘플 승인 전까지는 환불 가능합니다. 샘플 승인 후 본주문 안 하는 경우, 샘플은 배송해드리며 진행비는 환불 불가합니다. (샘플 제작 비용 발생)

7. **Q. 중국 공장이랑 소통 안 될까 봐 걱정돼요.**
   A. 저희가 중간에서 한국어로 소통합니다. 전담 PM이 배정되어 카톡 or 메일로 실시간 응대합니다.

8. **Q. MOQ가 얼마인가요? 100개만 가능한가요?**
   A. MOQ는 300개부터 가능합니다. 소량은 단가가 높아질 수 있으니 견적 문의 주세요.

9. **Q. 샘플만 먼저 받아볼 수 있나요?**
   A. 네. 샘플 제작 후 사진으로 확인 → 승인 → 실물 배송 가능합니다. (샘플 진행비 유료)

10. **Q. 3D 모델링 파일이 없는데 가능한가요?**
    A. 가능합니다. 앞/뒤면 이미지만 있어도 3D 샘플을 만들어드립니다. 손그림 or 실물 사진도 OK.

11. **Q. 색상이 다르게 나오면 어쩌죠?**
    A. PANTONE 색상 코드 or 실물 사진 기준으로 매칭합니다. 샘플 단계에서 색상 확인 후 승인하시면 동일하게 양산됩니다.

12. **Q. 포장은 어떻게 되나요?**
    A. OPP 개별포장 / 행택 추가 / 박스 벌크 중 선택 가능합니다. 브랜드 라벨 / 스티커 추가도 가능합니다.

13. **Q. 세금계산서 발행 가능한가요?**
    A. 네. B2B 거래 시 세금계산서 / 계약서 / 산출내역서 발행 가능합니다.

14. **Q. 디자인 수정은 몇 번까지 가능한가요?**
    A. 샘플 단계에서 2회까지 무료 수정 가능합니다. 3회 이상은 추가 비용 발생할 수 있습니다.

15. **Q. 긴급 제작 가능한가요?**
    A. 표준 50일보다 빠른 긴급 납기는 별도 상담 후 가능 여부 안내드립니다. (단가 상승 가능)

16. **Q. 샘플 진행비는 왜 받나요?**
    A. 샘플 제작 / 사양 확정 / 커뮤니케이션 착수 비용입니다. 진지한 고객에게 집중하기 위함이며, 본주문 시 전액 차감됩니다.

17. **Q. 다른 업체 견적과 비교해도 되나요?**
    A. 환영합니다. 단, 단가만 비교하지 마시고 '총비용(불량/지연 리스크 포함)'을 비교해보세요.

18. **Q. 키링 말고 다른 제품도 하나요?**
    A. 현재는 3D PVC 키링 전문으로 운영 중입니다. 다른 카테고리는 별도 문의 주세요.
